## Tutorial

Задаём переменные окружения для дальнейшей работы с терминалом и GitHub.
```bash
$ export GITHUB_USERNAME=Nikonnor
$ export GIST_TOKEN=<сохраненный_токен>
$ alias edit=subl
```

Создаём рабочую директорию с именем юзера, а в ней папку `/workspace`. Далее переходим в папку.
Командой `$ pwd` выводим полный путь к созданной директории.
Командой `$ cd ..` поднимаемся по дереву на 1 директорию вверх.
```sh
$ mkdir -p ${GITHUB_USERNAME}/workspace
$ cd ${GITHUB_USERNAME}/workspace
$ pwd
$ cd ..
$ pwd
```
Первая команда `pwd` возвращает: `/home/nikonnor/Nikonnor/workspace`

Вторая: `/home/nikonnor/Nikonnor`

Затем создаем директории для заданий, проектов и отчетов внутри ранее созданной директории.
```sh
$ mkdir -p workspace/tasks/
$ mkdir -p workspace/projects/
$ mkdir -p workspace/reports/
$ cd workspace
```

Устанавливаем Node.js, распаковываем архив, перемещаем в удобную папку и удалем архив.
```sh
$ wget https://nodejs.org/dist/v6.11.5/node-v6.11.5-linux-x64.tar.xz
$ tar -xf node-v6.11.5-linux-x64.tar.xz
$ rm -rf node-v6.11.5-linux-x64.tar.xz
$ mv node-v6.11.5-linux-x64 node
```

В следующем блоке посмотрим список файлов в установленной директории с Node.js, чтобы убедиться в том, что все правильно установилось, затем делаем файлы доступными глобально и созданиём скрипт активации, который активирует путь для Node.js всякий раз, когда он запускается.
```sh
$ ls node/bin
$ echo ${PATH}
$ export PATH=${PATH}:`pwd`/node/bin
$ echo ${PATH}
$ mkdir scripts
$ cat > scripts/activate<<EOF
export PATH=\${PATH}:`pwd`/node/bin
EOF
$ source scripts/activate
```

Скачиваем gist
```sh
$ gem install gist
```

Создаём файл `~/.gist` с токеном, доступным для чтения или записи только текущему пользователю.
```sh
$ (umask 0077 && echo ${GIST_TOKEN} > ~/.gist)
```

Клонируем репозиторий c первой лабораторной работой и создаем для него отдельную папку с отчетом. Далее редактируем и публикуем на GitHub Gist.
```sh
$ export LAB_NUMBER=01
$ git clone https://github.com/tp-labs/lab${LAB_NUMBER} tasks/lab${LAB_NUMBER}
$ mkdir reports/lab${LAB_NUMBER}
$ cp tasks/lab${LAB_NUMBER}/README.md reports/lab${LAB_NUMBER}/REPORT.md
$ cd reports/lab${LAB_NUMBER}
$ edit REPORT.md
$ gist REPORT.md
```
